# FlappyBird_DQN
使用DQN算法玩FlappyBird。环境使用PaddlePaddle + PLE + PARL。

# 安装依赖包
pip install gym

pip install paddlepaddle-gpu==1.6.3.post97

pip install parl==1.3.1

pip install pygame
